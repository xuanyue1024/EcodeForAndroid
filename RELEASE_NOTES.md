# Ecode Android 发布说明 v1.0.0-preview

## 📱 应用信息

**应用名称**: Ecode  
**版本号**: 1.0.0-preview  
**版本代码**: 1  
**最低 SDK**: Android 8.0 (API 26)  
**目标 SDK**: Android 14 (API 35)  
**构建日期**: 2026-01-29

## ✨ 主要功能

### 1. 用户认证系统
- **用户登录**
  - 支持用户名和密码登录
  - 密码采用 MD5 加密保护
  - 支持验证码验证（可选）
  - 登录状态自动保持（通过 Token）
  
- **服务器配置**
  - 自定义服务器地址
  - 支持 HTTP/HTTPS 协议
  - 配置信息本地持久化

### 2. 智能扫码
- **二维码/条形码扫描**
  - 基于 Google ML Kit 的高性能扫码引擎
  - 使用 CameraX 提供流畅的相机体验
  - 支持多种码制（QR Code、EAN、UPC 等）
  - 实时扫码预览和反馈

- **扫码确认**
  - 扫描结果展示
  - 一键提交扫码数据
  - 扫码历史追踪

### 3. 用户信息管理
- **个人信息展示**
  - 用户详细信息查看
  - 基于 WebView 的富文本展示
  - 响应式布局设计

- **账户管理**
  - 安全退出登录
  - Token 自动刷新

## 🚀 构建说明

### 构建发布版本

1. **清理项目**
   ```bash
   ./gradlew clean
   ```

2. **构建 Release APK**
   ```bash
   ./gradlew assembleRelease
   ```

3. **构建位置**
   - APK 文件: `app/build/outputs/apk/release/app-release.apk`

### 构建调试版本

```bash
./gradlew assembleDebug
```

### 运行测试

```bash
./gradlew test
```

## 📦 安装说明

1. 下载 `app-release.apk` 文件
2. 在 Android 设备上启用"未知来源"安装权限
3. 点击 APK 文件进行安装
4. 授予必要的权限（相机、网络）
5. 首次启动时配置服务器地址

## ⚙️ 配置说明

### 首次使用

1. **配置服务器地址**
   - 点击登录界面的"设置服务器"按钮
   - 输入服务器 URL（例如: `https://your-server.com`）
   - 保存配置

2. **登录账户**
   - 输入用户名和密码
   - 如需验证码，点击验证码图片刷新
   - 点击"登录"按钮

3. **使用扫码功能**
   - 从用户信息界面进入扫码页面
   - 授予相机权限
   - 对准二维码/条形码进行扫描
